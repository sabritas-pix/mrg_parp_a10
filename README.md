# MRG_PARP - Sistema de ordenes

## Descripción de la App
MRG_PARP es una aplicación móvil y web diseñada parala operación de taquerías y restaurantes. Se creó con el fin de llevar un mejor control entre los pedidos de las tres categorías: Pedidos en local, pedidos por teléfono y pedidos en aplicaciones.
Queremos mejorar la fluidez del trabajo y reducir tiempos.

## Framework
* **Framework:** Flutter
* **Arquitectura:** MVC (Modelo-Vista-Controlador)
* **Patrones de Diseño:** Singleton y Observer

## Metodología de Desarrollo
* **Metodología:** Scrum 
* **Duración del Sprint:** 2 semanas, en total 3 sprints.
